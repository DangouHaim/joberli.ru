# Joberli.ru

## Getting Started

### Requirements

You must have OpenServer or another local web server

#### Server configuration

HTTP : Apache-2.4+
PHP : PHP-5.5+
MySQL : MySQL-5.6+


### Installing

#### Local deploy

Clone project to working area (domains folder)

Import test database from file testdb.sql

#### Remote deploy

Clone project to working area (domains folder)

Create database

Replace .htaccess on .htaccess - release
